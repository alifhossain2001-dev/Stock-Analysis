# Stock Analysis Web Application

This project is a dynamic stock analysis web application built using HTML, CSS, and JavaScript.

## Overview
The application displays multiple stock listings with financial data including:
- Stock price
- Net income
- Market capitalization
- Dividends

The project uses HTML data attributes to store financial metrics and JavaScript to dynamically analyze and categorize stocks.

## Features
- Reads financial data using dataset attributes
- Dynamically classifies stocks as "cheap" or "expensive"
- Applies conditional styling using CSS classes
- Structured and modular code organization

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM manipulation)
- Data attributes

## Skills Demonstrated
- DOM traversal and manipulation
- Conditional logic implementation
- Dynamic class assignment
- Financial data interpretation
- Clean UI structuring
